# Bunk-Finder

Inspired by my friends frustrations about attendence I made tool that helps students determine how many more days they need to attend classes to reach the minimum required attendance (75%), and how many days they can safely take leave without dropping below the requirement.

## Inputs Required

Number of classes per day

Number of working days in the semester

Current attendance percentage

Total classes conducted so far

## Output

The application displays:

Number of days you must attend to reach 75% attendance

Maximum leave days allowed while maintaining 75%

Leave days taken so far
